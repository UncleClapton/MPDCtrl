# MPD-Ctrl

MPD Ctrl is a light-weight, desktop gadget-like, client app for the MPD (Music player daemon) http://www.musicpd.org/ . It should also work with MPD-compatible servers such as Mopidy https://www.mopidy.com/ .  

Requirement:  
Windows 10 ("Fall Creators Update/.NET Framework 4.7.1" or later) and a host running MPD.
